PhysIntuit: Physical Intuition Benchmark

A comprehensive benchmark for evaluating physical reasoning capabilities in vision-language models

View on GitHub

Overview

PhysIntuit is a novel benchmark designed to evaluate the physical reasoning capabilities of vision-language models. It features a diverse set of physical scenarios in both 2D and 3D environments, testing models' ability to understand and predict physical interactions.

Key Features

Comprehensive Coverage

Tests various physical phenomena including gravity, force, and collision across different complexity levels.

Dual Format Evaluation

Includes both binary classification tasks and state prediction challenges.

Multi-domain Testing

Evaluates physical reasoning in both 2D and 3D environments.

Dataset Composition

Our dataset includes a balanced distribution of different physical scenarios and interaction types, ensuring comprehensive evaluation of model capabilities.
